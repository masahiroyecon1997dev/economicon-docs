このガイドでは、Economiconをインストールして、最初の分析（OLS回帰）を実行するまでの手順を説明します。

## インストール

<Steps>

1. **インストーラーをダウンロードする**

   [GitHub Releases ページ](https://github.com/masahiroyecon1997dev/Economicon/releases) を開き、最新バージョンの `Economicon_x.x.x_x64-setup.exe` をダウンロードします。

2. **インストーラーを実行する**

   ダウンロードした `.exe` ファイルをダブルクリックします。

   :::caution[SmartScreen の警告について]
   本アプリは現在デジタル署名を行っていないため、Windows SmartScreen による警告が表示される場合があります。その場合は「詳細情報」→「実行」を選択してください。
   :::

3. **WebView2 ランタイム（初回のみ）**

   Microsoft Edge WebView2 が未インストールの場合、初回起動時に自動でダウンロード・インストールされます。インターネット接続が必要です。

4. **起動する**

   インストール完了後、デスクトップのショートカットまたはスタートメニューから Economicon を起動します。

</Steps>

## 最初の分析：CSVを読み込んでOLSを実行する

<Steps>

1. **CSVファイルを読み込む**

   画面上部の「ファイルを開く」ボタンをクリックし、分析したい CSV ファイルを選択します。

   読み込みが完了すると、データがテーブルとして表示されます。列名と行数を確認してください。

   :::tip
   手元に CSV ファイルがない場合は、「シミュレーション」機能でサンプルデータを生成できます。
   :::

2. **分析メニューを開く**

   上部のヘッダメニューから「推定」→「OLS（通常最小二乗法）」を選択します。

3. **変数を指定する**

   - **被説明変数（Y）**: 予測・説明したい変数（例: 賃金、売上高）
   - **説明変数（X）**: 影響を与えると考える変数（例: 教育年数、広告費）

   ドロップダウンから列名を選択します。

4. **実行する**

   「実行」ボタンをクリックします。結果が画面下部に表示されます。

5. **結果を確認する**

   | 項目 | 内容 |
   |---|---|
   | 係数 | 説明変数が1単位増えたときの被説明変数の変化量 |
   | 標準誤差 | 係数の推定精度 |
   | t 値 | 係数の有意性を測る統計量 |
   | p 値 | 帰無仮説（係数=0）が成立する確率。0.05 未満で有意とみなすことが多い |
   | 信頼区間 | 係数の95%信頼区間 |

6. **結果をコピーする（任意）**

   結果パネル右上のコピーボタンから、Markdown または LaTeX 形式でクリップボードにコピーできます。

</Steps>
