Economicon is a desktop app that helps users start econometric analysis through a GUI workflow.

## Current Implementation Status

- At this time, only **OLS (Ordinary Least Squares)** is implemented.
- Fixed effects, IV, GMM, DID, and RDD are not implemented yet.
- Progress will be updated on the releases page.

## OS Support

- The distributed app is currently **Windows only**.
- A macOS build is not available yet, and has not been verified.
- If you need macOS support, please send a request via GitHub Issues or X.
