Economicon is designed to lower the barrier to econometrics by reducing setup and coding friction.

## Why It Exists

Many learners struggle with environment setup and syntax before they can focus on model assumptions and interpretation. Economicon helps users start from data and output quickly.

## Positioning

Economicon is not a replacement for full code-based workflows. It is a practical entry point for learning, exploration, and early checks.

## OS Note

- Windows build is available.
- macOS is not available yet.
