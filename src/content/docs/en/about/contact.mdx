## Bug Reports and Feature Requests

For bug reports, troubleshooting, feature requests, and improvement ideas, please use GitHub Issues.

- It helps if you include your environment and the steps needed to reproduce the issue.
- Suggestions for useful new features are welcome.
- If a similar Issue already exists, adding details there is appreciated.

## Educational Distribution

Economicon is provided under the MIT License, so it is fine to distribute and use it in classes, seminars, workshops, labs, and training settings.

If you want to confirm anything about distribution or classroom use, please contact us through GitHub Issues.
