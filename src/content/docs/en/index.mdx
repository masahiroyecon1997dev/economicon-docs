## Key Features

<CardGrid stagger>
  <Card title="High-Speed Data Processing" icon="rocket">
    Powered by Rust and the Polars engine. Handles large datasets with ease and no lag.
  </Card>
  <Card title="Major Estimation Models" icon="puzzle">
    OLS, fixed effects, random effects, 2SLS/GMM, DID, RDD, and Heckman selection — all built in and ready to use.
  </Card>
  <Card title="Intuitive UI" icon="laptop">
    No coding required. Filtering, column operations, and simulation data generation are all mouse-driven.
  </Card>
  <Card title="Copy Results Instantly" icon="document">
    Copy regression output in Markdown or LaTeX format directly to your clipboard — paste straight into papers or reports.
  </Card>
</CardGrid>

## Installation

<Steps>
1. **Download**

   Get the latest `Economicon_x.x.x_x64-setup.exe` from the [GitHub Releases page](https://github.com/masahiroyecon1997dev/Economicon/releases).

2. **Run the Installer**

   Double-click the `.exe` file. If Windows SmartScreen shows a warning, click "More info" → "Run anyway".

3. **Launch**

   After installation, open Economicon from the desktop shortcut or the Start menu.
</Steps>

:::note
Currently **Windows only**. macOS and Linux support is not planned.
:::
