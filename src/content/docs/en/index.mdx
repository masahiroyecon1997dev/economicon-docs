## Installation

<Steps>
1. **Download**

   Get the latest `Economicon_x.x.x_x64-setup.exe` from the [GitHub Releases page](https://github.com/masahiroyecon1997dev/Economicon/releases).

2. **Run the Installer**

   Double-click the `.exe` file. If Windows SmartScreen shows a warning, click "More info" → "Run anyway".

3. **Launch**

   After installation, open Economicon from the desktop shortcut or the Start menu.
</Steps>

:::note
Currently **Windows only**. macOS and Linux support is not planned.
:::
