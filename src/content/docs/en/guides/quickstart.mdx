## Installation

<Steps>

1. **Download the installer**

   Open the GitHub Releases page and download the latest installer.

2. **Run the installer**

   Double-click the `.exe` file.

3. **Launch the app**

   Open Economicon from the desktop shortcut or Start menu.

</Steps>

## Uninstallation

<Steps>

1. **Close the app**

2. **Open Installed Apps in Windows settings**

3. **Select Economicon and uninstall**

4. **Optionally remove leftover files**

</Steps>
