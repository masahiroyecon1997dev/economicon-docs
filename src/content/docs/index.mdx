## 主な特徴

<CardGrid stagger>
  <Card title="高速なデータ処理" icon="rocket">
    Rust と Polars エンジンを搭載。大規模なデータセットでもストレスのない分析が可能です。
  </Card>
  <Card title="主要な推定モデル" icon="puzzle">
    OLS・固定効果・変量効果・2SLS/GMM・DID・RDD・Heckman 選択モデルなど、実証分析に必要なモデルを標準搭載。
  </Card>
  <Card title="直感的な UI" icon="laptop">
    プログラミング不要。フィルタリング・列操作・シミュレーションデータ生成もマウス操作で完結します。
  </Card>
  <Card title="結果をすぐにコピー" icon="document">
    分析結果を Markdown・LaTeX 形式でクリップボードにコピー。論文やレポートにそのまま貼り付けられます。
  </Card>
</CardGrid>

## インストール方法

<Steps>
1. **ダウンロード**

   [GitHub Releases ページ](https://github.com/masahiroyecon1997dev/Economicon/releases) から最新の `Economicon_x.x.x_x64-setup.exe` をダウンロードします。

2. **インストーラーを実行**

   ダウンロードした `.exe` ファイルをダブルクリックして実行します。SmartScreen の警告が表示された場合は「詳細情報」→「実行」を選択してください。

3. **起動**

   インストール完了後、デスクトップのショートカットまたはスタートメニューから起動します。
</Steps>

:::note
現在は **Windows 専用**です。macOS / Linux 対応は予定されていません。
:::
