Economiconは、計量経済学の初学者・学部生・大学院生・研究者が、コードを書かずにGUI操作で分析を始められるデスクトップアプリです。

## 重要: 現在の実装範囲

- 現在、推定モデルとして実装済みなのは **OLS（通常最小二乗法）のみ** です。
- 固定効果、IV、GMM、DID、RDD などは、現時点では未実装です。
- 今後の実装状況は [リリース情報](/economicon-docs/ja/reference/releases/) で更新します。

## OS対応状況

- 現在配布しているのは **Windows版** です。
- macOS版は、現状まだ作成しておらず、動作確認もできていません。
- macOS版が必要な場合は、GitHub Issue または X のメッセージで要望を送ってください。

## 主な機能

### データ入出力

- **対応形式**: CSV / Excel / Parquet のインポート・エクスポート
- **日本語対応**: 日本語ファイル名・列名・文字コード（Shift-JIS、UTF-8 等）に対応

### データ管理・加工

- テーブルの名前変更・複製・削除
- **不変性設計**: 操作は常に新テーブルとして保存。元データを上書きしない
- **Join / Union**: 複数テーブルの結合・連結
- **変数計算**: 数式を使って新しい列を生成
- **ラグ・リード列**: パネルデータ分析用のタイムラグ変数を生成
- **フィルタリング**: 列メニューから条件を指定してデータを絞り込み

### 統計解析

- **基本統計量**: 平均・中央値・分散・標準偏差・null 数など
- **相関行列**: 変数間の相関係数を一覧表示

### 推定モデル

すべての推定結果は**係数・標準誤差・t 値・p 値・信頼区間**をセットで表示します。

| モデル | 内容 |
|---|---|
| OLS | 実装済み |
| そのほかの推定モデル | 未実装（順次対応予定） |

### 結果のコピー

分析結果を以下の形式でクリップボードにコピーできます。

| 形式 | 用途 |
|---|---|
| テキスト | メモ・メール |
| Markdown | GitHub / Notion / Zenn |
| LaTeX | 学術論文・レポート |

### シミュレーション

- 統計モデル検証用のサンプルデータを自動生成
- シード値の手動指定で結果を再現可能

## 対象ユーザー

- 計量経済学を学び始めた学部生
- 卒論・修論でデータ分析を行う大学院生
- 素早くデータを探索したい実務研究者
- RやStateの前に手法を直感的に理解したい方
