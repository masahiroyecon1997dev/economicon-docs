## 現在の連絡先

- GitHub リポジトリ: [Economicon](https://github.com/masahiroyecon1997dev/Economicon)
- GitHub Issues: [Issues ページ](https://github.com/masahiroyecon1997dev/Economicon/issues)

## バグ報告と機能提案について

バグ報告、不具合の相談、機能提案、改善アイデアは、GitHub Issues の活用をお願いします。

- 再現手順や利用環境を一緒に書いてもらえると、確認しやすくなります。
- こんな機能があると便利、という提案も歓迎です。
- 既に同じ内容の Issue がある場合は、そちらに追記してもらえると助かります。

## 教育用途での配布について

Economicon は MIT ライセンスのもとで提供しており、授業、演習、ゼミ、勉強会、研修などで配布して利用していただいて問題ありません。

配布方法や授業での使い方について確認したいことがあれば、GitHub Issues から連絡してください。
