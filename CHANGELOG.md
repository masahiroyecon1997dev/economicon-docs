# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.4.0] - 2026-05-23

### ✨ Added

- **シミュレーションデータ作成に連番の選択肢を追加**: シミュレーションデータ生成機能に、連番オプションを追加。これにより、ユーザーはID列などの連番データを簡単に生成できるようになります。
- **パネル時間カラム追加機能**: パネルデータ分析のための時間カラムを追加する機能を実装。これにより、ユーザーはパネルデータの時間変数を簡単に管理できるようになります。
- **基本統計量の出力形式拡張**: 基本統計量もマークダウン形式、TEX、テキストで出力できる機能を追加。これにより、ユーザーは分析結果を様々な形式で利用できるようになります。
- **信頼区間の出力形式拡張**: 信頼区間の出力もマークダウン形式、TEX、テキストで出力できる機能を追加。これにより、ユーザーは信頼区間の結果を様々な形式で利用できるようになります。
- **サイドバーに分析結果表示**: 分析結果をサイドメニューに表示する機能を追加。これにより、ユーザーはテーブルと分析結果をスイッチングして確認できるようになります。また、テーブルのタブと分析結果のタブが並ぶUIになりました。
- **仮説検定**: 仮説検定の機能を追加。これにより、ユーザーは分析結果に基づいて仮説検定を行い、平均や分散の統計的な有意性を評価できるようになります。
- **画面構成の変更**: 各画面のフォームと分析結果をテーブルタブと同じ階層のタブに分けるように変更。これにより、ユーザーはフォームと分析結果を切り替えて確認できるようになります。
- **グループ別統計量機能** : グループ別の基本統計量のデータを作成する機能を追加。これにより、ユーザーはカテゴリ変数ごとに基本統計量を確認できるようになります。
- **プロットビュー機能**: データをプロットで表示する機能を追加。これにより、ユーザーは回帰係数や信頼区間を視覚的に確認できるようになります。
- **信頼区間シミュレーション機能**: α%（90%, 95%, 99%）信頼区間のサンプリングシミュレーション機能を追加。これにより、ユーザーはα%信頼区間の意味が何を示すかを理解しやすくなります。
- **回帰係数の漸近正規性シミュレーション機能**: 回帰係数の漸近正規性のシミュレーション機能を追加。これにより、ユーザーは回帰係数が大サンプルで正規分布に従うことを理解しやすくなります。
- **一致性シミュレーション機能**: 回帰分析の一致性のシミュレーション機能を追加。これにより、ユーザーは回帰分析の推定量がサンプルサイズが大きくなるにつれて真の値に近づくことを理解しやすくなります。
- **不偏性シミュレーション機能**: 回帰分析の不偏性のシミュレーション機能を追加。これにより、ユーザーは回帰分析の推定量の期待値が真の値と等しいことを理解しやすくなります。
- **ファイル削除機能**: インポート画面のゴミ箱アイコンよりファイル削除が可能になりました。

### 🗑 Deleted

- **列複製**: 列複製機能を削除。列編集関連の機能は列が新たに追加されるため、列複製の必要性が低くなったため削除することにしました。

### 🐞 Fixed

- **初回起動時の言語設定**: 初回起動時（設定ファイル作成時）にOSの言語設定を読み取って、初期設定の言語を自動で切り替えるように修正
- **ダイアログのアニメーション**: ダイアログが閉じるときにアニメーションなく消えてしまう問題を修正。

### 🛠 Internal

- **分析エンジンのベンチマーク追加**: 仮想的なデータとWooldridge や AER などの標準データセットを用いた計算整合性テストを実装

## [0.3.0] - 2026-04-05

### ✨ Added

- **信頼区間推定の追加**: 回帰分析の結果に信頼区間を表示する機能を追加
- **列移動機能の追加**: 列をドラッグ＆ドロップで移動できるようにする機能を追加
- **回帰分析の結果のコピー機能の追加**: 回帰分析の結果をtext, markdown, latex形式でクリップボードにコピーする機能を追加

### 🐞 Fixed

- **言語設定の不具合修正**: 言語設定が反映されない問題を修正
- **計算のエラー修正**: 計算機能の構文エラーをわかりやすいように修正
- **UI/UX の改善**:
  - ダイアログにスクロールを表示するよう修正
  - ラグ・リード列の追加のグループ化列を選択方式に変更
  - テーブル操作メニューと列操作メニューを右クリックで開くように修正

[0.3.0]: https://github.com/masahiroyecon1997dev/Economicon/releases/tag/v0.3.0

## [0.2.0] - 2026-03-29

### ✨ Added

- **相関行列作成**: 相関行列データを作成する機能を追加。
- **基本統計量の拡充**: `null数`, `null割合`, `有効サンプル数`, `母分散` を追加。
- **シミュレーションの再現性**: シード値を手動で指定できるオプションを追加。
- **データフィルタリング**: 列メニューからデータをフィルタリングできる機能を追加。

### 🐞 Fixed

- **UI/UX 改善**: 高速レスポンス時のローディングの「ちらつき」を解消。
- **エラーハンドリング**: サーバー起動失敗時にダイアログが出ない問題を修正。

### 🔒 Security

- **安定性向上**: 依存パッケージを最新バージョンに更新。

[0.2.0]: https://github.com/masahiroyecon1997dev/Economicon/releases/tag/v0.2.0

## [0.1.0] - 2026-03-24

### ✨ Added

- **📥 データインポート/エクスポート**
  - **多形式対応**: CSV, Excel, Parquet 形式のインポートにネイティブ対応。
  - **データ保存**: 解析済みデータを CSV, Excel, Parquet 形式で出力・保存する機能。
- **🗂️ データマネジメント**
  - **基本操作**: データセットの名前変更、複製、削除機能。
- **🛠️ データ加工・生成**
  - **テーブル操作**: 複数のデータを統合する `Join`（結合）および `Union`（連結）機能。
  - **シミュレーション**: 統計モデル構築のためのテスト用データ生成機能。
  - **変数操作**: 既存の列に基づいた高度な変数計算機能。
- **📊 統計解析**
  - **基本統計量**: 平均、分散、中央値などの記述統計量の算出。
  - **推定モデル**: 回帰分析 (OLS等) の実行機能。
- **⚙️ システム設定**
  - **外観カスタマイズ**: ダークモード/ライトモードのテーマ切り替え。
  - **国際化 (i18n)**: 複数言語でのインターフェース表示をサポート。
  - **エンコーディング**: 日本語データ等で重要なファイル文字コード設定の変更機能。
